# `tenplusgood.github.io`

这是一个可直接部署到 GitHub Pages 的个人学术主页静态站点。

## 已包含内容

- 个人简介
- 教育背景
- 研究方向与项目概述
- 论文与软件著作权
- 荣誉奖项与竞赛经历
- 联系方式

## 本地预览

直接用浏览器打开 `index.html` 即可预览。

## 部署到 GitHub Pages

1. 在 GitHub 上创建仓库 `tenplusgood.github.io`
2. 将本目录下文件上传到仓库根目录
3. 提交并推送到默认分支
4. 访问 `https://tenplusgood.github.io/`

## 后续建议

- 把你的个人照片替换掉当前页面中的文字头像
- 给论文补上 DOI、PDF、代码仓库或项目主页链接
- 如果你有 Google Scholar、ORCID、ResearchGate，也可以补到首页按钮区域
- 如果你想要更接近参考站点的 Jekyll 版本，后续可以在此基础上再迁移
